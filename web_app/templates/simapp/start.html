What is GENZ Thrusters?

This particle simulator was built to optimize the geometry of a rocket thruster. This problem we identified and decided to solve was a unique but cumulative common interest in the dynamics and launching of rockets. We wanted to determine the optimal form of a rocket thruster in terms of geometry and efficiency of the nozzle design. There are many launches that take place over the world from SpaceX's Starlink to NASA's Mars Exploration Rover that have inspired us to wonder at what can be achieved in optimizing the most plausible and efficient rocket thruster there can be.

Why does GENZ Thrusters incorporate a Genetic Algorithm and a Particle Simulator?

A genetic algorithm is a simple algorithm which allows us to create a very complex object with very little apriori knowledge of how the system works. Concretely, this means that without knowing about what makes a thruster a good one versus a bad one, we are able to iteratively generate a progressively better thruster. All we have to specify is the physics of the problem (i.e. we just need to calculate the thrust from a particular thruster).

To do this, we use a simple custom particle simulator. Each particle is a high energy particle which is supposed to mimic the conditions of a hot gas (the fuel of a typical rocket thruster). We easily calculate the thrust generated from a particular geometry of thruster, and use this to define how well a thruster is doing.

How to Use GENZ Thrusters?

Check out our code at https://github.com/sxrodrigues/physhackathon2020 and http://genzthrusters.pythonanywhere.com/
Installing pybind11:
pip3 install pybind11
Installing the simulator module (this must be done in the root directory of this repository):
pip3 install .
You can test the code is running and this should print the value 3 to the console.
python3 src/main.py
Running the Web Application in your browser as a local host.
pip3 install Django
Then enter the "web_app" directory and run:
python3 manage.py runserver

Future Ideas for the project

  • Optimizing further not just the nozzle of the thruster but as well as the complete geometry of the thruster.
  • Adding performance optimizations such as multi-threading.
  • Adding fields and intermolecular forces. Atmospheric particles is a possibility as well.
